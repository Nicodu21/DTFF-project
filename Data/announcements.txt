## annoucments.xlsx Data Frame Description 

Number of Rows: 184

Number of Columns: 3

#Columns:

'Company': Contains the company ticker (e.g., TSLA, NVDA, META).
Data Type: String
Format: Company ticker (e.g., TSLA, NVDA)

'Date': Date of the announcement in YYYY-MM-DD format.
Data Type: Date
Format: YYYY-MM-DD (e.g., "2014-01-30")

'Announcement': Details about the product or event announced (e.g., "Model S P85D").
Data Type: String
Format: Description of the announcement.

#Exemple sturcture

Company     Date         Announcement
TSLA        2014-01-30   Model S P85D
NVDA        2014-03-08   NVDA GeForce GTX 800M Series
META        2014-04-30   Facebook Paper
AMZN        2014-06-18   AMZN Fire Phone
NVDA        2014-09-02   GeForce GTX 980 and GTX 970
